css BOX MODEL


  • 경계선 밖의 여백 영역, 다른 요소와의 여백 지정에 사용
  • 경계선
  • 경계선과 컨텐츠 사이의 영역
  • 표시되는 컨텐츠 영역

css 박스 모델은 html문서를 구성하는 요소들을 브라우저에 그려서 나타느낸데 일괄적으로 적용되는 모델입니다.

html문서의 구성요소들을 단계적으로 감싸고 있는 형태로 구성되어 있으며 기존 요소 두가지는 경계선을 나타내는 border와 실제 출력 내용을 나타내는 content입니다.

border바깥영역을 margin border content 사이 영역을 padding이라고 합니다.

이 div 영역에는 box-mobel에 따라 margin: top 10px, right 20px, bottom 40px, left 20px이 적용되었고
padding 상하좌우 모두 25px
border 20px의 녹색 실선으로 설정되었습니다.

이 div의 크기는 가로 세로 300 X300 으로 설정되었습니다.
